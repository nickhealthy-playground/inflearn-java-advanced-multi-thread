# 프로젝트 지시사항

## README.md 업데이트 규칙

커밋 요청 시 또는 새 코드 작성 후, **반드시** `README.md`의 학습 기록 섹션을 업데이트할 것.

### 업데이트 내용

새로 추가된 패키지/클래스마다 아래 항목을 기록:

- **패키지 경로** (`src/thread/...`)
- **핵심 클래스/인터페이스** 목록
- **학습 개념** (한 줄 요약)
- **주요 API** (사용된 메서드, 필드)

### 형식

`README.md`의 `## 학습 기록` 섹션 아래에 다음 구조로 추가:

```markdown
### `패키지명` — 학습 주제
- `ClassName` — 한 줄 설명
- 핵심 개념 요약 (1~2줄)
- 주요 API: `method()`, `field`
```

새 항목은 **가장 위에** 추가 (최신 순).
